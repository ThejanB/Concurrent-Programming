
How ro Run ->

    1. Compilation
        - javac .\TheSleepingBarber.java

    2.Run 
        2.1 Run with default values - 
            - java .\TheSleepingBarber
        2.2 Run with custom values
            - java TheSleepingBarber [customerMeanSeconds] [haircutMeanSeconds] [maxWaitingCustomers] [numberOfBarbers]
                
                eg: java .\TheSleepingBarber.java 10 15 3 2
		            java .\TheSleepingBarber.java 5 15 5 3

    3. Termination
        Press Ctrl + C to terminate the simulation
